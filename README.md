# Cab Booking Application 🚕

A robust, user-friendly Cab Booking application designed to streamline urban transportation. This platform connects passengers with nearby drivers, offering real-time tracking, fare estimation, and seamless ride management.

---

## 🚀 Features

### For Passengers
* **Real-Time Booking:** Find and book nearby cabs instantly.
* **Fare Estimation:** Get upfront pricing based on distance, traffic, and vehicle type.
* **Live Tracking:** Track your ride's location and ETA in real-time.
* **Ride History:** View past trips, receipts, and driver details.

### For Drivers
* **Driver Dashboard:** Toggle availability status (Online/Offline).
* **Ride Requests:** Accept or reject incoming ride requests with route details.
* **Navigation:** Built-in route optimization for pickups and drop-offs.
